# Personal Password Creator

A simple Python application that generates unique and personalized passwords based on user information such as their name, favorite food, favorite animal, and lucky number. The project demonstrates Python fundamentals, user input handling, string manipulation, and randomization.

## Features

- Generates personalized passwords
- Uses random special characters for stronger passwords
- Creates multiple password suggestions
- Easy-to-use command-line interface
- Beginner-friendly Python project

## Technologies Used

- Python 3
- `random` module

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/varsharma07/Personal-Password-Creator.git
```

2. Navigate to the project folder:

```bash
cd Personal-Password-Creator
```

3. Run the program:

```bash
python password_generator.py
```

## Example

```text
===== Personalized Password Generator =====

Enter your name: Var
Enter your favorite food: Pizza
Enter your favorite animal: Tiger
Enter your lucky number: 07

Here are your personalized passwords:

1. Var@Pizza07
2. Pizza#Tiger07
3. Tiger$Var07
4. VarPizza!07
5. Pizza07%Tiger
```
